Lab 13 - Loops

Challenge

The challenge of this assignment is to create a structured web page that integrates a JavaScript implementation of the "FizzBuzz" function, styled with custom CSS, and dynamically outputs results to a webpage using jQuery. It involves writing a JavaScript function to handle the FizzBuzz logic with multiple conditions (3, 5, 7), appending results to the DOM, and debugging issues to ensure functionality both locally and on the server.

Problems

I didn't really have any problems with this lab. Following the lecture which showed how to make the fuction, it was easy to implement in the lab.
